<h1>Breadcrumb</h1>
<p>

</p>
<h4>Usage</h4>
<ul>
  <li>
    A series of clickable links that show the user the path they took to get to
    the current page.
  </li>
  <li>
    Alternative way to navigate webpages, specifically for deeper pages within
    the IA
  </li>
  <li>
    A useful feature for search engine optimization (SEO), as it provides
    additional context to search engines about the structure of a website.{" "}
  </li>
</ul>
